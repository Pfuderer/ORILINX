Release Notes
===============================

v1.0.1
-----------------

* ORILINX takes a fasta file of mammalian DNA as input and produces a csv or bed file indicating the probability that each locus can act as an origin of replication.
* Documentation is available at https://orilinx.readthedocs.io.
* Released with Pfuderer PL, Berkemeier F, Nassar J, Crisp A, Jaworski JJ, Moore J, Sale JE, Boemo MA. A genome language model for mapping DNA replication origins. bioRxiv 2026. [`Read on bioRxiv <https://doi.org/10.64898/2026.01.29.702604>`_].
